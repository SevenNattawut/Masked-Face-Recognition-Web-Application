About this project

Introduction
The Masked Face Recognition with Machine Learning in the Web Application project aims to develop a facial recognition system that can recognize and identify individuals, even when they are wearing masks. This is important as the COVID-19 pandemic has led to widespread mask-wearing, which can make it difficult for existing facial recognition systems to work properly.
The goal of this project is to create a web application that can be used in communal areas, such as apartments or offices, where a facial recognition system is used for security purposes. This would allow individuals to pass the scanning process while wearing masks, making it more convenient and safe for everyone involved.
How it works
The camera will capture the video and try to detect the people face in each frame. It will compare the cropped face image with the model to find the name from the confidence percentage. Then, display the name, confidence percentage, and highlight the face on the screen. And record to the user timeline table at the same time.
Technologies and Tools

Python

Python is an object-oriented high level programming language. It is used to develop websites, build automated software, or analyze data. With its simple syntax and vast libraries, it allows programmers to achieve many tasks with ease.

Tensorflow

TensorFlow is an open source software library for machine learning and deep learning purposes. It is notably used to build and train machine learning models which include facial recognition.

Keras

Keras is a deep learning API running on the TensorFlow platform. It is mainly used to implement neural networks. Notable networks are provided in its methods, namely VGG and ResNet.

Scikit-Learn

Scikit-learn is a Python library providing tools and algorithms for machine learning and modeling which covers: regression, classification, clustering, and model selection.


OpenCV

OpenCV is an open source software library built for computer vision and machine learning. The library contains a lot of optimized algorithms used for a variety of tasks, for example, facial detection and recognition, object detection, movement tracking, and similar image search.

Flask

Flask is the Python-based web framework library which is good for small scale web applications. It can be used to create the backend side of the application, including APIs and can run a small trafic scale local server for development.

HTML

HTML, or HyperText Markup Language, is a markup language for documents designed to be displayed on web browsers.

CSS

CSS, or Casacding Style Sheets, is a styling sheet language used for describing elements presentation on documents used markup languages such as HTML, or XML.


Bootstrap

Bootstrap is a CSS framework used to enhance web design and functionality. It offers pre-designed templates and components that can be easily customized with CSS.

JavaScript

JavaScript, shorten as JS, is a programming language that is one of the core technology of the World Wide Web. It is what makes many websites become interactive.

jQuery

jQuery is a JavaScript framework used to simplify web development by providing pre-built functions and methods. It allows for easier manipulation of HTML documents and enables the creation of interactive web pages by selecting specific web elements and triggering JavaScript functions on them.

MySQL

MySQL database is a relational database that stores the data in the form of tables that connect to each other by using keys.
